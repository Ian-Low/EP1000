For project management, we are using GitHub where we upload our website and updates to.
First off is to create a new repository for us to upload our work into.



We have to give the repository a name followed by setting it to private or public.



Than you can start by uploading your html code or any coding language you have learned.



When uploading files (updating the repository), you have to label your commit. I label mine by using the date I upload any new commits. Also putting a comment with what I have updated.
